# Calculate append=skip

В этой директории размещены шаблоны утилит Calculate 2.2. Большинство утилит
поддерживает более новый формат шаблонов. Директория с шаблонами осталась для
поддержки пакетов утилит calculate-builder и calculate-assemble. Для настройки
других пакетов утилит, используйте шаблоны версии 3.1.
